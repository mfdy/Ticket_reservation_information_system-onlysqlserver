# 机票预定信息系统 —— 数据库原理大作业


## 注意

本系统基于MS SQL SERVER2018 实现，仅仅包含数据库设计部分，不具有完整操作能力

## 特点

1. 本系统满足BCNF
2. 含有单表查询和多表连接查询
3. 创建了完整性约束和自定义约束
4. 具有多个触发器
4. 进行了数据可的备份和恢复处理
6. 有权限分配和角色设置
